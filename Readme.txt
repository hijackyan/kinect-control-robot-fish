运行环境：VS2010
操作说明：
2D_fish
        举起单手超过一秒 项目（游戏）开始。
	开始后单手举起为暂停，开始后举起双手一秒为结束。
        畅游模式：
   	手向左右伸或扭头——不同档位的拐弯	
	手向前伸——不同档位的加速	
	踢腿——摆尾
	扭屁股——全速前进。

        竞速模式：
        双手伸出做出握方向盘状——开启竞速模式
	双手握方向盘状并左右扭动方向盘——左右拐弯
	跺左右脚——减速、加速。
        放下双手——模式结束

3D_fish
        举起单手超过一秒 项目（游戏）开始。
	开始后单手举起为暂停，开始后举起双手一秒为结束。
      畅游模式：
        头向左右摆动——带速拐弯
        原地右转——原地右拐弯
 	原地左转——原地左拐弯
	向上跳起——上浮，蹲下——下潜
	向前伸手——不同档位加速	
	后退一步——鱼向后退。
	双手做出拍照状——截取鱼上摄像头图像。

        竞速模式：
        双手伸出做出握方向盘状——开启竞速模式
	双手握方向盘状并左右扭动方向盘——左右拐弯
	跺左右脚——减速、加速。
        放下双手——模式结束
